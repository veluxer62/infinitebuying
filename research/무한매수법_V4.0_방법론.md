# 무한매수법 V4.0 방법론

> 원문: https://cafe.naver.com/infinitebuying/79263

## 개요

무한매수법은 **일반모드**와 **소진모드**로 구성된다.

- **일반모드**: 소진 전, 쿼터매도를 포함한 일련의 과정
- **소진모드**: T값이 1회분 미만으로 남은 시점부터, 특정 조건을 만족하여 일반모드로 되돌아가기 전까지의 과정

### 추천 종목

| 종목 | 특징 | 추천 대상 |
|------|------|----------|
| TQQQ | 변동성 낮음, 수익 낮음 | 초심자 필수 시작 |
| SOXL | 변동성 높음, 수익 높음 | TQQQ 익숙 후 병행 |

> **오피셜**: TQQQ, SOXL 이외 다른 종목은 추천하지 않음

## 1. 원금 설정 및 분할 설정

### 원금 설정
- 무한매수법 전용 원금을 별도로 설정
- 매수 후 잔금이 남아도 **반드시 다른 투자에 사용 금지**

### 분할 수 선택

| 분할 수 | 성향 | 1회 매수금 (원금 2만달러 기준) |
|--------|------|-------------------------------|
| 20분할 | 공격적, 위험 | $1,000 |
| 30분할 | 중간 | ~$667 |
| 40분할 | 방어적, 안정적 | $500 |

### 복수 운용 시 주의사항
- TQQQ + SOXL 동시 운용: 각 원금을 독립적으로 설정, 한 계좌에서 진행 가능
- 동일 종목의 다른 분할 수 동시 운용: **반드시 다른 계좌**로 분리

## 2. T값 (Turn) 정의

**T = 진행 회차**를 의미하며, `1.0T`는 1회분이 진행된 상태.

### T값 변화 규칙

| 이벤트 | T값 변화 |
|--------|---------|
| 1회 매수 | +1 |
| 절반 매수 | +0.5 |
| 쿼터매도 | 직전T × 0.75 |
| 지정가매도 후 LOC 매수 (1회) | × 0.25 + 1 |
| 지정가매도 후 LOC 매수 (절반) | × 0.25 + 0.5 |

**예시:**
- T=7 에서 1회 매수 → T=8
- T=7 에서 절반 매수 → T=7.5
- T=7 에서 쿼터매도 → T=7×0.75=5.25

> T값은 소수점 제한 없이 그대로 표현 (예: 5.165479525)

## 3. 별지점 (★) 정의

**별지점 = 매수와 매도를 구분하는 기준가격** (V2.2부터 도입)

### 별% 계산 공식

| 종목/분할 | 별% 공식 |
|----------|---------|
| 20분할 TQQQ | (15 - 1.5T) % |
| 40분할 TQQQ | (15 - 0.75T) % |
| 20분할 SOXL | (20 - 2T) % |
| 40분할 SOXL | (20 - T) % |

### 별지점 계산

```
별지점 = 평균단가 × (1 + 별%)
```

**예시** (20분할 SOXL, 평단 $38.30, T=8.6):
```
별% = (20 - 2×8.6)% = 2.8%
별지점 = 38.30 × (1 + 2.8%) = $39.37
```

### 전반전 vs 후반전

| 구분 | T 범위 (40분할 기준) | 별% | 특징 |
|------|---------------------|-----|------|
| 전반전 | T=0 ~ 19.999... | 양수 (+) | 평단 위에서 매도 기회 |
| 후반전 | T=20 ~ 39 | 음수 (-) | 평단 아래에서 손절 시도 |

### 매수/매도점 조정
- **매수점**: 별지점 - $0.01 (겹침 방지)
- **매도점**: 별지점 그대로 사용

## 4. 매수금 설정

### 1회 매수금 공식

| 분할 | 1회 매수금 |
|------|-----------|
| 20분할 | 잔금 ÷ (20 - T) |
| 40분할 | 잔금 ÷ (40 - T) |

**예시** (원금 $20,000, 40분할):
- 1회차 매수금: $20,000 ÷ 40 = $500
- 실제 사용액 $478 → 잔금 $19,522
- 2회차 매수금: $19,522 ÷ 39 = $500.56

> T 변화(+1, +0.5, ×0.75) 관계없이 동일 공식 적용 → 매일 매수금이 미세하게 변동

## 5. 매수 방법

> **원칙**: 매수는 모두 LOC(시장가 주문)만 사용, 주간장은 이용하지 않음

### (1) 처음 매수 (T=0, 보유량=0)

- 무조건 1회 전액 매수 시도
- 전날 종가보다 **10~15% 위** 값으로 LOC 매수 설정

**예시** (TQQQ 종가 $45.93):
```
큰수: $51.44 (약 12% 위)
LOC 매수: $51.44에 12개
LOC 매수: $47.53에 1개
LOC 매수: $44.13에 1개
...
```

### (2) 전반전 매수 (T=1 ~ 분할수/2 미만)

- 1회 매수금의 **절반**: 별지점 LOC 매수
- 1회 매수금의 **나머지 절반**: 평단 LOC 매수
- 큰 하락 대비용 추가 LOC 매수를 아래에 배치

**예시** (별지점 $78.12, 평단 $69.75):
```
LOC 매수: $78.11에 3개  ← 별지점-0.01
LOC 매수: $69.75에 4개  ← 평단
LOC 매수: $67.40에 1개  ← 하락 대비
LOC 매수: $59.91에 1개  ← 하락 대비
...
```

### (3) 후반전 매수 (T=분할수/2 ~ 분할수-1)

- 1회 매수금 **전체**를 별지점 LOC 매수로 시도
- 큰 하락 대비용 추가 LOC 매수를 아래에 배치
- T > 분할수-1 이면 **소진모드로 전환**

**예시** (별지점 $59.55):
```
LOC 매수: $59.54에 9개  ← 별지점-0.01
LOC 매수: $56.85에 1개  ← 하락 대비
LOC 매수: $51.68에 1개  ← 하락 대비
LOC 매수: $47.37에 1개  ← 하락 대비
...
```

## 6. 매도 방법

> **원칙**: 전반전/후반전 구분 없이 동일하게 적용

### 쿼터매도 (LOC 매도)
- 보유 수량의 **1/4**을 별지점에 LOC 매도

### 최종매도 (지정가 매도)
- 나머지 **3/4** 수량을 지정가 매도

| 종목 | 지정가 매도 기준 |
|------|----------------|
| TQQQ | 평단 대비 **+15%** |
| SOXL | 평단 대비 **+20%** |

### 매도 타이밍
- 지정가매도: 프리장 시작 시 설정
  - 서머타임: 저녁 **5시**
  - 비서머타임: 저녁 **6시**
- 효력: 프리장 → 본장 → 애프터장 전체

**예시** (별지점 $59.55, 보유 141개):
```
LOC 매도:  $59.55에 35개  ← 보유량의 1/4
지정가 매도: $75.76에 106개 ← 나머지 3/4
```

---

## 7. 종료 시점

1. **보유 수량 = 0** → 1회전 종료
2. 지정가매도 후 LOC 매수가 발생한 경우 → 종료가 아니므로 계속 진행
3. 종료 후 처음 매수로 재시작

### 원금 재설정 방식 선택

| 방식 | 설명 |
|------|------|
| 복리 | 원금 + 수익금을 다음 회전 원금으로 설정 |
| 단리 | 원래 원금으로만 재시작 |

> 라오어 운용 방식: 1년간 복리로 진행 후 원금 재설정

## 8. 큰 하락장 대응 (큰수 매수)

증권사는 현재가 대비 **약 ±20% 초과** 주문을 거부하는 시스템이 있다.

3배 레버리지 특성상 큰 하락 시 이 제한에 걸릴 수 있으므로, **큰수 매수** 개념을 활용하여 우회한다.

> 자세한 내용: https://cafe.naver.com/infinitebuying/53077

## 흐름도 요약

```
원금 설정 및 분할 수 결정
        ↓
T=0, 처음 매수 (현재가+10~15%로 LOC)
        ↓
┌───── 전반전 (T < 분할수/2) ─────┐
│  매수: 절반은 별지점, 절반은 평단   │
│  매도: 1/4 쿼터(별지점 LOC)       │
│        나머지 지정가(+15/20%)      │
└──────────────────────────────────┘
        ↓
┌───── 후반전 (분할수/2 ≤ T < 분할수) ─────┐
│  매수: 전체를 별지점 LOC                   │
│  매도: 동일 (쿼터 + 지정가)                │
└──────────────────────────────────────────┘
        ↓
T ≥ 분할수-1 → 소진모드 (별도 문서 참조)
        ↓
보유량 = 0 → 종료 → 재시작
```

| 별% (40분할 TQQQ) | (15 - 0.75T)% |
| 별% (20분할 SOXL) | (20 - 2T)% |

| 매수점 | 별지점 - $0.01 |

| T (1회 매수) | T + 1 |
| T (절반 매수) | T + 0.5 |
| T (쿼터매도) | T × 0.75 |
